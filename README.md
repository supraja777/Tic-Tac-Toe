# Project Title
=======
Tic Tac Toe Game

## Project Overview
---------------
This project implements a Tic Tac Toe game with a Python script for pipeline management and an HTML-based game interface with JavaScript logic.

## Technology Stack
-----------------
* Python
* HTML
* JavaScript

## File Structure
----------------
| File | Description |
| --- | --- |
| app.py | Initializes and prints the status of a completed pipeline. |
| index.html | Provides the HTML structure and layout for the Tic Tac Toe game board, navigation, and user interaction elements. |
| script.js | Implements the game logic for the Tic Tac Toe game, handling player turns, win conditions, and game reset. |

## How to Run/Install
---------------------
1. Open the terminal and navigate to the project directory.
2. Run the Python script using `python app.py` to test the pipeline status.
3. Open the `index.html` file in a web browser to play the Tic Tac Toe game.
