# Tic Tac Toe Game
## Project Overview
This project implements a Tic Tac Toe game using HTML, CSS, and JavaScript, with a supplementary Python script for pipeline management.
## Technology Stack
* HTML
* CSS
* JavaScript
* Python
## File Structure
| File | Description |
| --- | --- |
| index.html | Provides the HTML structure for the Tic Tac Toe game |
| style.css | Styles the game application |
| script.js | Implements the game logic for the Tic Tac Toe game |
| app.py | Initializes and prints the status of a completed pipeline |
## How to Run/Install
1. Open `index.html` in a web browser to play the Tic Tac Toe game
2. Run `app.py` using Python to initialize and print the pipeline status
